abstraction layer

IMAGE SEARCH ABSTRACTION LAYER

"Generally, an abstraction layer is an API (set of functions) which is hiding the underlying implementation/complexity. The purpose is to provide a clear and easier-to-use interface over an existing set of functionality." (stackexchange user)

This is the 4th fcc API backend project.

    User story:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Try me!

http://localhost:3000/api/search/panda

http://localhost:3000/api/search/dragonfly


with offset:

http://localhost:3000/api/search/panda?offset=6

http://localhost:3000/api/search/dragonfly?offset=10

You can change the name of these request and change or remove offset with the ones of your choice.


get latest search:

http://localhost:3000/api/latest
